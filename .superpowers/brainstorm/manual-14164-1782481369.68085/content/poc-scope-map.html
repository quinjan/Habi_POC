Habi POC Scope Map

The POC should prove one sharp thing before Habi becomes an MVP: that private project history can produce useful purchasing intelligence faster than the current manual workflow.

Possible POC center of gravity
A

Private Project Memory

Ingest 2-3 completed projects, normalize materials/services/suppliers, then search and retrieve evidence from past work.

B

Recommendation Starter Pack

Given one new warehouse project profile, suggest likely materials, services, supplier references, missing scope questions, and confidence levels.

C

Quote / Supplier Comparison

Compare supplier options using historical price, bundle coverage, availability, delivery risk, and selection reasoning.
