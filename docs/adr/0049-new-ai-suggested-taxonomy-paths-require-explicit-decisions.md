# New AI-Suggested Taxonomy Paths Require Explicit Decisions

When AI suggests a category path that would create any new project taxonomy node, Habi will require an explicit Taxonomy Decision before the affected approved candidate can become importable. Top-Level Category and subcategory suggestions use the same gate level in the POC: existing taxonomy paths require no gate, while new AI-suggested paths must be approved, mapped, or rejected. Manual reviewer category edits remain candidate-specific review corrections and do not create remembered AI-suggestion defaults.
