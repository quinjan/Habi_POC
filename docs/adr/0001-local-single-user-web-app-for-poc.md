# Local Single-User Web App For POC

The Habi POC will be built as a local single-user web app with a real backend, data store, file storage, review workflow, and search/retrieval layer, but without SaaS concerns such as authentication, teams, permissions, billing, deployment hardening, or multi-user collaboration. This keeps the proof focused on AI parsing across source types, human-reviewed import, evidence-backed project memory, and project-scoped AI search accuracy while avoiding premature product infrastructure.
