# Top-Level-Only Suggestions Remain Unresolved

An AI suggestion with only a Top-Level Category is not an importable Resolved Category Path, even when that Top-Level Category already exists in the selected Project Workspace. The review response should show an unresolved Taxonomy Gate that communicates a subcategory is required. If the path would create new project taxonomy, it requires the same explicit Taxonomy Decision as any other new AI-suggested taxonomy path, but the affected candidate remains unresolved for import until the reviewer also selects, creates, or maps to a subcategory.
